# monddns

强大易用的ddns脚本

## 🤔为什么选择monddns

- ✅支持多个账号的多个域名与多个ip间的绑定

- ✅支持单子域名多A/AAAA记录

- (未来将)支持多个服务商
  
  - ~~好吧这事实上取决于我本人使用什么服务商~~
  
  - ~~主要是lua的sdk真没人给啊~~

- ✅支持多种ip获取方式，支持一次获取多个ip
  
  - 未来还可以方便地扩展

- ✅依赖精简，按需加载

## ☁️目前支持的服务商

- Cloudflare
- NameSilo
- 阿里云

## ✌️目前支持的ip获取方式

- http/https

- 命令获取

- 固定值

## 📦依赖

**环境依赖：** Lua5.1或者更高(但是仅在lua5.1和lua5.4上进行测试)

**全局依赖：** `cjson`、`LuaSocket`
